# Project Memo: Amazon Chronos Bolt Exploration
Date: 2026-06-17

## Current Status
Successfully implemented a series of Proof of Concept (PoC) and backtesting scripts using the `amazon/chronos-bolt-base` model on Apple Silicon (MPS).

## Accomplishments
1.  **Initial PoC (`chronos_poc.py`):**
    *   Verified model functionality with mock data.
    *   Established MPS acceleration setup.
2.  **AAPL Backtest (`chronos_backtest.py`):**
    *   Tested on 6 months of real Apple stock data.
    *   Results: Avg MAE ~$5.46, Directional Accuracy ~44.44%.
3.  **BTC-USD Deep Backtest (`chronos_deep_test.py`):**
    *   Tested on 1 year of Bitcoin data with 60-day context.
    *   Multi-horizon evaluation (5-day and 10-day).
    *   Results (5-day): Avg MAPE 2.92%, Directional Accuracy **58.33%**.
    *   Generated visualization: `btc_backtest_results.png`.

## File Manifest
- `chronos_poc.py`: Basic setup and mock test.
- `chronos_backtest.py`: Rolling window backtest for stocks.
- `chronos_deep_test.py`: Comprehensive multi-horizon backtest for crypto/high-volatility assets.
- `btc_backtest_results.png`: Visual performance chart.
- `docs/plan-v2.md`: Original PoC plan.
- `docs/plan-backtest.md`: Backtest implementation plan.
- `docs/plan-deep-test.md`: Deep evaluation plan.
